一个基于vue.js的组件库 (A Component Library Based on vue.js)
## 安装
``` bash
npm install v-plugs-ayu --save
```
在man.js中引入即可
``` javascript
import ayu from 'v-plugs-ayu'
Vue.use(ayu)
```
